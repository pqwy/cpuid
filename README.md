Cpuid allows detection of CPU features from OCaml.

Cpuid is distributed under the ISC license.

## Installation

cpuid can be installed with `opam`:

    opam install cpuid

If you don't use `opam` consult the [`opam`](cpuid.opam) file for build
instructions.
